# Choice...Repeat PWA v2

This is a cleaner, more mobile-first revision of the PWA prototype.

## Changes in v2
- Single-column app-like layout
- Much less wasted space
- Stronger visual hierarchy
- Better mobile scaling
- Compact stat cards
- Better card styling
- Quick-fill chips for the coach
- History cards instead of a wide table

## Run locally
```bash
python3 -m http.server 8000
```

Open:
```text
http://localhost:8000
```

## Deploy to GitHub Pages
Upload the files to the root of a GitHub repo and enable:
Settings → Pages → Deploy from branch → main / root
